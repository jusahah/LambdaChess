About LambdaChess

What is it? What does it do?

LambdaChess is a personal project to utilize Amazon Lambda computation service for scalable chess analysis. It analyses chess games and displays visualizations of move evaluations.

What Lambda stands for?

Amazon allows Stockfish 7 binary be executed on its Lambda cloud service. This way LambdaChess can take in PGN game, split it into positions, and send each position separately to Lambda computation process. Sending is done from Node.js server by HTTP.

On the Amazon side Node.js receives the position, and sends it to separate local Stockfish 7 process to be analysed. Stockfish churns for a while, spits out the analysis and job is done.

After all positions are analysed, positions are joined back together forming a PGN text file, which is then routed to end user.

Note that as each position is analysed on separate Stockfish 7 instance, positions arrive back in semi-random order. You can see the results dripping back to you in real time. When the last position has arrived back you are presented with full PGN file.

Is it free?

For now, yes. Note that Amazon Lambda computation service is not free. It is pretty cheap though, and analysis of a single game (40 moves, default ply depth of 16) costs fractions of one euro cent. Analysing one thousand games is still well within Amazon Free Tier-package.

Note that ply depth of 16 is modest. Probably comparable to ELO 2600 or something. Think it more of a blunder check than real analysis. Later on I may add possibility of buying deeper analysis. Something like 24 ply, which is already way above Carlsen strength.
